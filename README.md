**Web Sostenible GreenTech**
  Nombre: Alba Durán Bernal
  ------------------------------------------------
  ### 1. ¿Qué pasaba con la web original?
  La página anterior era un desastre para el planeta por varias razones:
  -Pesaba demasiado: Usaba librerías gigantes solo para poner un botón verde.
  -Gastaba mucha batería: Al tener código innecesario, los móviles tenían que trabajar de más y se calentaban.
  -Consumía muchos datos: Obligaba al usuario a gastar su tarifa de datos en descargar cosas que no servían para nada

  ### 2. ¿Cómo lo he arreglado? 
 -He borrado todos los frameworks pesados 
 
 -Ahora el navegador no tiene que leer miles de líneas de código extra.
 
-He configurado las imágenes para que solo se descarguen si el usuario llega a verlas, lo que se llama lazy loading

 -He quitado scripts que solo servían para poner la fecha, usando una sola línea de código simple

### 3. ¿Por qué esto ayuda al medio ambiente?
Mi código ahora respeta la Economía Circular mediante un ahorro de energía, como la web pesa menos, los servidores y las redes gastan menos electricidad al enviarla.Además
al ser un código ligero, los teléfonos antiguos no sufren. Esto frena la obsolescencia programada (que la gente tire su móvil porque va lento) y genera menos basura electrónica.
